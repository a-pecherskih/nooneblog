## Небольшой сайт - статейник (API) 

### Выполнение команд

- composer install
- создание .env из .env.example (прописать подключение к бд) 
- php artisan key:generate
- php artisan migrate
- php artisan db:seed
- php artisan test

### Команда для обновления количества лайков и просмотров
- php artisan article:update-counts
